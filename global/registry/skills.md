# Skills Registry

Available slash commands:

## Meta-Skills (Self-Extension)
| Skill | Purpose |
|-------|---------|
| /init | Initialize Marvin project config in the current directory |
| /new-agent | Scaffold a new specialized agent |
| /new-skill | Scaffold a new skill/slash command |
| /new-rule | Scaffold a new domain knowledge rule |
| /audit-agents | Audit codebase technologies and identify gaps in agent coverage |

## Universal Skills
| Skill | Purpose |
|-------|---------|
| /research | Deep research on any topic using web search + Exa + Context7 |
| /review | Code review with focus on quality, security, and best practices |
| /spec | Start an OpenSpec Spec-Driven Development workflow |
| /ralph | Start a Ralph Loop for long-running autonomous tasks |
| /remember | Save something to Marvin's persistent memory (global or project) |
| /meta-prompt | Generate an optimized prompt for a specific task |

## Data Engineering Skills
| Skill | Purpose |
|-------|---------|
| /pipeline | Design and scaffold a complete data pipeline (ingestion → transformation → serving) |
| /dbt-model | Generate dbt models with proper layering, tests, and documentation |
| /dag | Generate Airflow DAGs from a description with best practices built in |
| /data-model | Design dimensional data models (star schema, snowflake schema, Data Vault) |

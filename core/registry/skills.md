# Skills Registry

## Meta-Skills
- `/init` — Bootstrap a .claude/ directory with CLAUDE.md, memory, registry, rules, and settings for a new project
- `/new-agent` — Scaffold a new specialized agent
- `/new-skill` — Scaffold a new skill/slash command
- `/new-rule` — Scaffold a new domain knowledge rule
- `/audit-agents` — Scan a codebase for technologies in use and identify gaps in agent coverage
- `/handoff-reference` — Full handoff protocol reference with examples

## Universal
- `/research` — Deep research (web search + Exa + Context7)
- `/review` — Code review (quality, security, best practices)
- `/spec` — Plan, specify, implement, and verify features through structured phases with atomic commits
- `/ralph` — Run a long autonomous task across multiple context windows using filesystem checkpointing
- `/remember` — Save to Marvin's persistent memory
- `/meta-prompt` — Generate an optimized prompt for any task, agent, or skill using meta-prompting techniques
- `/tdd` — RED-GREEN-REFACTOR test-driven development cycle
- `/debug` — Systematic root cause analysis: reproduce, isolate, fix, verify
- `/adr` — Document architecture decisions in standard ADR format

## Utility
- `/worktree` — Manage git worktrees for parallel isolated development

## Data Engineering
- `/pipeline` — Design and scaffold a complete data pipeline
- `/dbt-model` — Generate dbt models with tests and docs
- `/dag` — Generate Airflow DAGs from a description
- `/data-model` — Design dimensional data models
